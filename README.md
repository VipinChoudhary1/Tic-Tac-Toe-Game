# Tic-Tac-Toe-Game
A classic Tic Tac Toe game created using HTML, CSS, and JavaScript. Enjoy this fun browser-based game with a clean and responsive design. Perfect for two players to challenge each other!
<b>Features</b>
2 Player Gameplay: Play with a friend on the same device.
Responsive Design: Optimized for both desktop and mobile browsers.
Interactive and Minimal UI: Clean visuals and intuitive gameplay.
Game Logic: Automatic winner detection and tie identification.
<b>How to Play</b>
Open the game in any modern web browser.
Take turns clicking on the grid to place your mark (X or O).
Align 3 of your marks in a row, column, or diagonal to win.
If all squares are filled without a winner, the game declares a tie.
