# Tic-Tac-Toe-Game
A classic Tic Tac Toe game created using HTML, CSS, and JavaScript. Enjoy this fun browser-based game with a clean and responsive design. Perfect for two players to challenge each other!
<br>
<b>Features</b><br>
2 Player Gameplay: Play with a friend on the same device.<br>
Responsive Design: Optimized for both desktop and mobile browsers.<br>
Interactive and Minimal UI: Clean visuals and intuitive gameplay.<br>
Game Logic: Automatic winner detection and tie identification.<br>
<b>How to Play</b><br>
Open the game in any modern web browser.<br>
Take turns clicking on the grid to place your mark (X or O).<br>
Align 3 of your marks in a row, column, or diagonal to win.<br>
If all squares are filled without a winner, the game declares a tie.<br>
